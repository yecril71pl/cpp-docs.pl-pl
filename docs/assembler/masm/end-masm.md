# <a name="end"></a>END

Oznacza koniec modułu i opcjonalnie ustawia punkt wejścia programu na *procId*.

## <a name="syntax"></a>Składnia

> **End** ⟦*procId*⟧

## <a name="remarks"></a>Uwagi

Argument *procId* jest prawidłowy tylko w 32-bitowym MASM.
