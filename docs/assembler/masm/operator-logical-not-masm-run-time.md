# <a name="operator--masm-run-time"></a>operator! (W czasie wykonywania MASM)
Logiczna negacja. Używana tylko wewnątrz [. Jeśli](../../assembler/masm/dot-if.md), [. GDY](../../assembler/masm/dot-while.md), lub [. Powtórz](../../assembler/masm/dot-repeat.md) blokuje i oceniane w czasie wykonywania, nie w czasie zestawu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
!  
expression  
  
```
