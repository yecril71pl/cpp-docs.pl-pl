# <a name="operator-ptr"></a>operator PTR
Pierwszy wymusza operator *wyrażenie* powinien być traktowany jako mających określony `type`. Drugi operator określa wskaźnik `type`.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
