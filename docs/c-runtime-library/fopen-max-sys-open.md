## <a name="remarks"></a>Uwagi

Jest to maksymalna liczba plików, które można otwierać jednocześnie. `FOPEN_MAX` jest to nazwa zgodne z ANSI. `_SYS_OPEN` zapewnia zgodność z istniejącym kodem.
