# <a name="c26460-use_const_reference_arguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  Argument odwołania "% argument%" dla funkcji "% Function%" można oznaczyć jako `const`. Zobacz [ C++ podstawowe wytyczne con. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
