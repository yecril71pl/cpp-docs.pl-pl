# <a name="cbookmarkoperator-"></a>CBookmark::operator =
Przypisuje `CBookmark` obiektu do innego.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      CBookmark& operator =(   
   const CBookmark& bookmark    
) throw( );  
```  
  
## <a name="remarks"></a>Uwagi  
 Ten operator jest wymagana tylko w **CBookmark\<0 >**.
