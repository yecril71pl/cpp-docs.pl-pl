# <a name="compiler-error-c2022"></a>C2022 błąd kompilatora
"numer": zbyt duży dla znaku  
  
 Liczba ósemkowa po ukośnik odwrotny (\\) w znaku lub stała ciągu jest zbyt duży do reprezentowania znak.
