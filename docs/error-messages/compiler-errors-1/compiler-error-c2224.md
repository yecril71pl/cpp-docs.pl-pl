# <a name="compiler-error-c2224"></a>C2224 błąd kompilatora
po lewej ".identifier" musi mieć typ struct/union  
  
 Argument po lewej kropki (.) nie jest klasy, struktury lub związku.  
  
 Ten błąd może być spowodowany przez lewy operand, która jest zmienną Niezdefiniowany (w związku z tym wpisz `int`).
