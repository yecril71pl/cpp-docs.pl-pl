# <a name="compiler-error-c2277"></a>C2277 błąd kompilatora
'Identyfikator': nie można pobrać adresu funkcji członkowskiej  
  
 Nie można przyjąć adresu funkcji członkowskiej.  
  
 Poniższy przykład generuje C2277:  
  
```  
// C2277.cpp  
class A {  
public:  
   A();  
};  
(*pctor)() = &A::A;   // C2277   
```
