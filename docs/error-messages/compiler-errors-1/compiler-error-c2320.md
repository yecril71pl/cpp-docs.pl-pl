# <a name="compiler-error-c2320"></a>C2320 błąd kompilatora
Oczekiwano ":" aby występował po specyfikatorze dostępu "specyfikatora"  
  
 Słowo kluczowe `public`, `protected`, lub `private` musi być z dwukropkiem.
