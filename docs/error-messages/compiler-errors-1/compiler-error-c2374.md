# <a name="compiler-error-c2374"></a>C2374 błąd kompilatora
"identyfikator": zmiana definicji; Inicjowanie wielu  
  
 Identyfikator zainicjowano więcej niż raz.  
  
 Poniższy przykład generuje C2374:  
  
```  
// C2374.cpp  
// compile with: /c  
int i = 0;  
int i = 1;   // C2374  
int j = 1;   // OK  
```
