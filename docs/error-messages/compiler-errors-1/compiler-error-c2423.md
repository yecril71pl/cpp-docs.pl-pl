# <a name="compiler-error-c2423"></a>C2423 błąd kompilatora
"numer": niedozwolony skali  
  
 Kod zestawu wbudowanego używa wielu innych niż 1, 2, 4 lub 8 można skalować do rejestru.  
  
 Poniższy przykład generuje C2423:  
  
```  
// C2423.cpp  
// processor: x86  
int main() {  
   _asm {  
      lea EAX, [EAX*3]   // C2423  
      lea EAX, [EAX+EAX*2]   // OK  
   }  
}  
```
