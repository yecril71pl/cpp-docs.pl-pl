# <a name="compiler-error-c2806"></a>Błąd kompilatora C2806

"operator operatora" ma zbyt wiele parametrów formalnych

Przeciążony operator ma za dużo parametrów.

Poniższy przykład generuje C2806:

```cpp
// C2806.cpp
// compile with: /c
class X {
public:
   X operator++ ( int, int );   // C2806 more than 1 parameter
   X operator++ ( int );   // OK
} ;
```
