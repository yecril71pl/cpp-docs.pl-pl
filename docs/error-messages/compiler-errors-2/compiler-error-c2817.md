# <a name="compiler-error-c2817"></a>C2817 błąd kompilatora
typem zwracanym dla "operator delete" musi być "void"  
  
Przeciążone [operatora delete](../../standard-library/new-operators.md#op_delete) funkcja nie może zwracać wartości.
