# <a name="compiler-error-c2996"></a>Błąd kompilatora C2996

"Function": definicja szablonu funkcji cyklicznej

Definicja funkcji próbuje utworzyć wystąpienie klasy głównej szablonu. Tworzenie wystąpień szablonu cyklicznego jest niedozwolone.
