# <a name="compiler-error-c3610"></a>C3610 błąd kompilatora
"valuetype": typ wartościowy musi być "boxed" przed można wywołać metody "method"  
  
 Domyślnie typ wartości nie jest na stercie zarządzanej. Przed wywołaniem metody z klasy środowiska wykonawczego platformy .NET, takie jak `Object`, musisz przenieść typu wartości na stercie zarządzanej.  
  
 C3610 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
