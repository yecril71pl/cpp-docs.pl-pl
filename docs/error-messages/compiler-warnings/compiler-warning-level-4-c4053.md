# <a name="compiler-warning-level-4-c4053"></a>Kompilator ostrzeżenie (poziom 4) C4053

jeden operand void dla "?:"

`?:` Operator otrzymuje wyrażenie typu `void`. Wartość `void` operand jest niezdefiniowana.
