# <a name="nmake-warning-u4006"></a>Ostrzeżenie NMAKE U4006
Niezdefiniowane makra specjalnego: "makra"  
  
 Nazwa danego makra specjalnego nie jest zdefiniowana i rozszerza na wartość nothing.
