# <a name="project-build-error-prj0021"></a>Błąd PRJ0021 kompilacji projektu
Narzędzie "narzędzia" właściwości "property" zawiera nieprawidłową nazwę pliku.  
  
 Nazwa pliku określona we właściwości, ***właściwości***, narzędzia, ***narzędzia***, jest nieprawidłowy.  
  
 Użyto nieznanego lub nieprawidłowego makra.  
  
 Zobacz [Ustawianie właściwości projektu Visual C++](../../ide/working-with-project-properties.md) informacji na temat sposobu dostępu do właściwości projektu.
