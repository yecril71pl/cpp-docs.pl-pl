# <a name="debugging-support-classes"></a>Klasy obsługi debugowania

MFC udostępnia następujące klasy, które ułatwiają Debugowanie problemów z alokacją pamięci dynamicznej.

[CDumpContext](../mfc/reference/cdumpcontext-class.md)<br/>
Udostępnia miejsce docelowe zrzutów diagnostycznych.

[CMemoryState](../mfc/reference/cmemorystate-structure.md)<br/>
Struktura, która zapewnia migawki użycia pamięci. Używany również do porównywania wcześniejszych i późniejszych migawek pamięci.
