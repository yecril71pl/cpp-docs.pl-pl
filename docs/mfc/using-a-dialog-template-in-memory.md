# <a name="using-a-dialog-template-in-memory"></a>Używanie szablonu okna dialogowego w pamięci

Zamiast korzystać z metod przedstawionych w [tabeli tworzenia okna dialogowego](../mfc/creating-a-dialog-class-with-code-wizards.md), można utworzyć dowolny rodzaj okna dialogowego pośrednio z poziomu szablonu okna dialogowego w pamięci. Aby uzyskać więcej informacji, zobacz Klasa [CDialog](../mfc/reference/cdialog-class.md) w *dokumentacji MFC*.
