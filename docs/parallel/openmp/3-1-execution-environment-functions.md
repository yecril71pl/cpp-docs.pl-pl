# <a name="31-execution-environment-functions"></a>3.1 Funkcje środowiska wykonawczego
Funkcji opisanych w tej sekcji mają wpływ na i monitorować wątków, procesorów i środowisko równoległe:  
  
-   **omp_set_num_threads** funkcji.  
  
-   **omp_get_num_threads** funkcji.  
  
-   **omp_get_max_threads** funkcji.  
  
-   **omp_get_thread_num** funkcji.  
  
-   **omp_get_num_procs** funkcji.  
  
-   **omp_in_parallel** funkcji.  
  
-   **omp_set_dynamic** funkcji.  
  
-   **omp_get_dynamic** funkcji.  
  
-   **omp_set_nested** funkcji.  
  
-   **omp_get_nested** funkcji.
